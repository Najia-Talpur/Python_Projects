# Pepe's Piri Piri Restaurant Ordering System using Python
This project is a simple console-based restaurant ordering system written in Python.  
It simulates a basic ordering system similar to a fast-food restaurant like Pepe's.

## Features
- Displays a restaurant menu with prices
- Allows customers to place orders
- Lets users choose flavour options
- Option to select a meal (includes fries and drink)
- Choose dips
- Supports multiple orders
- Automatically calculates the final bill

## Technologies Used
- Python
- Dictionaries
- Lists
- Functions
- Loops
- Conditional statements

## How It Works
1. The program displays the menu.
2. The customer selects a food item.
3. The customer chooses a flavour.
4. The customer decides if they want the item as a meal.
5. If it is a meal, the customer selects fries and a drink.
6. The customer selects a dip.
7. The order is stored in a list.
8. The program asks if the customer wants to order more items.
9. When finished, the program prints the full order receipt and calculates the total bill.

## Pricing Logic
- Each menu item has a base price.
- If the customer selects a meal, £1.80 is added to the item price.
